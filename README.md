# AI INSTALLER Landing Page

A modern, industrial-style landing page for AI INSTALLER - Technical Circular & Automated Systems.

## Features

- Bold, technical blueprint-inspired design aesthetic
- Fully responsive layout using Tailwind CSS
- Custom typography with Work Sans and Inter fonts
- Material Symbols icons
- Interactive demo section
- High-contrast, accessibility-focused design

## Tech Stack

- HTML5
- Tailwind CSS (via CDN)
- Google Fonts (Work Sans, Inter)
- Material Symbols icons

## Deployment

This site is optimized for deployment on Vercel.

### Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/ai-installer-landing)

Or manually:

```bash
vercel
```

## Local Development

Simply open `index.html` in your browser or use a local server:

```bash
python -m http.server 8000
# or
npx serve
```

Then visit `http://localhost:8000`
